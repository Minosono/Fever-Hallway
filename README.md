# Fever-Hallway

**Fever-Hallway** ist ein interaktives Video-Erlebnis (FMV Walking Simulator) im Browser.

## Über das Projekt

Dieses Projekt entstand im Rahmen eines **Schulprojekts im Kunst Projektkurs der 13. Klasse**. Es verbindet klassisches Film-Storytelling mit der Interaktivität von Videospielen. Der Spieler bewegt sich durch flure, interagiert Anrufern und trifft Entscheidungen, die den Verlauf und das Ende des Spiels beeinflussen.

## Technische Details

- **Entwicklung:** Web-Technologien (HTML5, CSS3, Vanilla JavaScript) ohne externe Frameworks.
- **Video Assets:** Die 3D-Umgebungen und Szenen wurden mit der **Unreal Engine** erstellt.
- **Post-Production:** Der Videoschnitt und die Bearbeitung erfolgten in **DaVinci Resolve**.

## Warnung

> [!WARNING]
> **Hinweis zu Audio-Inhalten:** Einige der verwendeten Stimmen und Dialoge in diesem Projekt wurden mit Hilfe von **Künstlicher Intelligenz (KI)** generiert.

## Starten des Projekts

Öffnen Sie einfach die `index.html` Datei in einem modernen Webbrowser, um das Spiel zu starten.

Oder starten Sie das Spiel direkt über die GitHub Page: [https://minosono.github.io/Fever-Hallway/](https://minosono.github.io/Fever-Hallway/)
